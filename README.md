# Ultimate-Dex
App desarrollada con los lenguajes Kotlin y Jetpack Compose. Se uso la PokeAPI en vez de bd para mostrar los pokemons.

-------------------------------------------

###### Arquitectura
- MVVM

###### Diseno
- Jetpack Compose

###### Images
- Pixel 6 pro y pixel 7 pro

###### otras dependencias agregadas
- Couroutines
- Compose Navigation
- Hilt Navigation
- Accompanist Swipe Refresh
